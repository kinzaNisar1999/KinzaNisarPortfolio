3D Animated Movie (FYP)

Role: Technical Artist

1. Technical Pipeline Overview

For my Final Year Project (Grade A), I co-produced a complete story-driven short film. Instead of traditional pre-rendering, we utilized Unity's Universal Render Pipeline (URP) to achieve near-real-time cinematic fidelity. My primary focus was establishing a robust character and asset pipeline between Daz 3D, Blender, and Unity.

2. Key Technical Contributions

  • Character Pipeline: Engineered a workflow using Daz 3D to generate age-variant morphs (Child vs. Adult) while maintaining consistent facial identity.
  • Cloth Simulation: Designed custom outfits in Marvelous Designer, simulating realistic fabric physics, and optimized mesh weights in Blender before Unity import.
  • Environment & Lighting: Constructed immersive scenes in Unity URP, utilizing a mix of baked lightmaps for static geometry and real-time lighting for characters.
  • Audio-Visual Sync: Manually synchronized voice-overs with facial animations directly within the Unity Timeline for precise lip-sync.

Heavy Truck USA

Role: Unity Developer

1. Project Overview

I served as a core developer for this heavy cargo simulation game, designing specialized game modes that required unique handling logic. My primary focus was on vehicle physics stability for heavy loads (10+ tons) and dynamic terrain traversal.

2. Key Mechanic: Trailer Hitching System

I engineered a physics-based coupling system where the truck must manually align with the trailer's kingpin. Upon connection, a ConfigurableJoint is instantiated to link the rigidbodies, and the truck's mass/drag values are dynamically updated to simulate the massive added load.

3. Key Feature: Animal Transportation Mode

  • Live Cargo Logic: Designed a unique mode where players transport livestock. I implemented logic where rough driving (sudden braking/turning) agitates the animals, causing a score penalty or fail state.
  • Vehicle Physics: Adjusted suspension stiffness specifically for this mode to handle the shifting center of mass from moving live cargo.
